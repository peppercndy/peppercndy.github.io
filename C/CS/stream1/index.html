三，state bit

state bit作用：描述流(stream)的状态

1. 四种state bit

标志 描述
goodbit 没有错(值为0)
eofbit 到达末尾
failbit
badbit something bad happend

2. fail与bad区分

fail:

  • type mismatch : 比如想要int类型，但是流中没有int
  • 文件无法打开
    bad:
  • 文件不能读到缓冲区(例如你想要读的文件被删了)
